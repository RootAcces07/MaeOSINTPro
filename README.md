# MaeOSINTPro
MaeOSINT est un scanner OSINT polyvalent : analyse WHOIS, DNS, ports, réseaux sociaux et sites sensibles. Il exporte tout en JSON propre pour l’audit, l’enquête et la cybersécurité. Simple, rapide et efficace pour collecter des infos en profondeur.

## Compatibilité 

Linux & Windows 

## Installation

```bash
git clone https://github.com/RootAcces07/MaeOSINTPro.git
```
```
cd MaeOSINTPro
```
```
python MaeOSINTPro.py
```
